# Oil Analysis Example — Fevereiro/2026

## Função deste documento
Exemplo de análise setorial/macro para commodities de energia. Útil como referência de profundidade, estrutura de raciocínio e mapa de variáveis.

## Avaliação honesta
O material é forte em:
- visão de tape + curva + inventories + geopolitics
- separação entre cenário base, bear e bull
- tradução para produtores brasileiros via USD/BRL
- identificação de gatilhos que mudam a tese

O material é fraco em:
- excesso de densidade para uso recorrente de comitê
- mistura entre análise do mercado global de petróleo e recomendação implícita de equities sem bridge suficiente
- confiança excessiva em números datados que envelhecem rápido
- pouca explicitação do que é fato, inferência e julgamento

## Regra de uso
Este documento não deve ser reutilizado como “verdade sobre petróleo”. Deve servir como:
- template de raciocínio
- biblioteca de variáveis importantes
- exemplo de como conectar macro global ao P&L de nomes brasileiros

## Estrutura recomendada para futuros updates de oil
1. Setup do mercado
2. Curva e estoques
3. Oferta
4. Demanda
5. Geopolítica
6. Cenários de preço
7. Tradução para BRL e equities locais
8. Implicação de portfólio
