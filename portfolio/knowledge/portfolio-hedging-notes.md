# Portfolio Hedging Notes

## Lições do tema regime shift
- hedge bom não precisa depender de uma tese única estar perfeitamente certa
- ouro pode funcionar mais como diversificador macro do que como aposta em superciclo amplo
- exposição a emergentes deve ser seletiva, não um bloco cego
- tech de qualidade não deve ser descartada apenas porque ficou cara; o problema quase sempre é sizing e preço pago
- caixa é opcionalidade quando o regime é incerto
