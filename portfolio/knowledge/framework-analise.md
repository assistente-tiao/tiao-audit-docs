# Framework de Análise

Referência metodológica do Comitê de Investimentos. Deve orientar análises, mas não substituir julgamento.

## Regra de uso
- consultar antes de análises estruturadas
- adaptar ao tipo de ativo, setor e disponibilidade real de dados
- sinalizar explicitamente quando algum bloco não se aplica ou não possui dados confiáveis

## Regra editorial crítica
Este framework é um **sistema de apoio**, não um ritual burocrático. Se um campo não acrescenta decisão, o analista deve resumir. Se um campo for crucial, deve aprofundar.
