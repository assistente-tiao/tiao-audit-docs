# Mapeamento do Portfólio

Documento vivo de composição, cobertura e status das teses. Deve ser atualizado pelo CIO sempre que houver mudança material.

## Regra crítica
Este arquivo não deve misturar dado estático com conclusão opinativa sem separar claramente as duas coisas.

## Fonte de verdade operacional
- composição e preços médios: planilha viva / base operacional
- ownership analítico: `portfolio/committee/coverage-map.md`
- teses resumidas: `portfolio/knowledge/thesis-registry.md`
- alertas de concentração e fatores: `portfolio/knowledge/portfolio-risk-map.md`

## Observação editorial
O material de origem tinha boa utilidade como fotografia inicial, mas estava inchado e misturava:
- composição
- cobertura
- concentração
- backlog analítico
- opinião do CIO

Esses blocos foram separados para facilitar manutenção.
