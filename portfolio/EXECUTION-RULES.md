# Execution Rules — Portfolio Intelligence Agent

## Objective
Construir o Portfolio Intelligence Agent seguindo estritamente o backlog do Kanban.

## Definition of Done
Uma task só conta como concluída quando:
- houve artefato real criado ou alterado
- o critério de aceitação do card foi atendido
- o status do card foi atualizado no Kanban

## Operating Rules
- O Kanban é a fonte de verdade para execução.
- Não inventar escopo fora dos cards.
- Progresso material = card mudando de status com entrega verificável.
- Cards devem andar na ordem de dependência definida.
- Antes de iniciar execução pesada, validar acessos externos críticos.
