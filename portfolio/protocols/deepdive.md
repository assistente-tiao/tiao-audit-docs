# Protocolo — Deep Dive

Quando uma posição específica for pedida:
- o analista responsável produz o relatório principal
- seguir `portfolio/knowledge/framework-analise.md` e `portfolio/templates/deepdive.md`
- incluir explicitamente cenário base / bull / bear quando fizer sentido
- incluir kill switch da tese
- incluir impacto cruzado macro e interação com outras posições do portfólio
- incluir teste de consistência macro e concentração de fator
- o CIO adiciona parecer final consolidando implicação para o portfólio
