# Protocolo — Update Macro

## Estrutura
1. panorama macro objetivo
2. principais mudanças desde o último update
3. impactos por bloco do portfólio
4. riscos de segunda ordem
5. implicações do CIO para alocação, sizing e prioridade de monitoramento
6. quando houver tese temática forte, testar explicitamente caso a favor, caso contra e custo de estar cedo demais

## Regra
Macro não é comentário solto; precisa terminar em implicação prática para o portfólio.
O uso recorrente desse protocolo deve seguir a cadência definida em `portfolio/ops/operating-cadence.md`.
