# Portfolio Intelligence Agent

Projeto OpenClaw para monitoramento de portfólio via Google Sheets, atualizações diárias de mercado e briefing dominical no Discord.

## Fonte de verdade operacional
- Kanban do VidClaw para execução
- Google Sheets para posições e dados operacionais

## Escopo V1
- posições atuais como input primário
- PnL consolidado em BRL
- atualização diária pós fechamento do mercado mais tardio
- briefing dominical com macro, posições, releases e implicações para carteira
