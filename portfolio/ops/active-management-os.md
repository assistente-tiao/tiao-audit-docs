# Active Management Operating System

## Sequence
1. live panel shows current posture
2. trigger system detects meaningful change
3. cadence determines when synthesis happens
4. CIO decision layer converts evidence into action

## Current operational stack
- portfolio state: `live-position-panel.csv`
- trigger logic: `trigger-taxonomy.md` + `trigger-matrix.csv`
- cadence: `operating-cadence.md`
- decisions: `cio-decision-log.csv` + `cio-decision-workflow.md`

## Principle
The committee should not improvise its process every week. It should improvise only its judgment when new facts require it.
