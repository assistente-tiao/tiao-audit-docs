# Monitoring Triggers

## Purpose
Translate portfolio monitoring into explicit triggers with clear escalation.

## Files
- taxonomy: `portfolio/ops/trigger-taxonomy.md`
- matrix: `portfolio/ops/trigger-matrix.csv`

## Core rule
A trigger is not an alert for its own sake.
A trigger exists to force one of three responses:
- observe
- review
- decide
